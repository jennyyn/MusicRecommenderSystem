# Chat
## Setup
1. Clone the repository: `https://github.com/jennyyn/MusicRecommenderSystem.git`
2. Get API key from [OpenAI Website](https://platform.openai.com/docs/overview)
3. `export API_KEY="your=key"`
5. Run `Main.java`

## Features
- Creative writing mode
- Professional writing mode
- Academoc writing mode
- Save/Load sessions

## Design Paterns
- Strategy: Different writing modes
- Factory: Request creation
- Observer: UI updates
